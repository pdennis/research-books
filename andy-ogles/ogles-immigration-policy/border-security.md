# Border Security and Enforcement

### Summary

> * Andy Ogles has consistently supported hardline border security measures, including completing the U.S.-Mexico border wall and calling for the border to be "closed down." ([Tennessean](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:65TG-NDS1-DXGX-616F-00000-00&context=1519360), [Tennessee Lookout](https://tennesseelookout.com/2022/08/05/get-dc-out-of-tennessee-says-republican-congressional-nominee-ogles/), [Daily Herald](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:5PK3-TVC1-JBCN-40BT-00000-00&context=1519360))

([Daily News Journal](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:66V2-XKJ1-DXGX-637M-00000-00&context=1519360), [Tennessean](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:661Y-RYH1-JB1V-G3GP-00000-00&context=1519360))

([Jackson Sun](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:6B2B-JMY1-DXGX-6021-00000-00&context=1519360), [Tennessean](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:6B24-KBR1-JB1V-G3FK-00000-00&context=1519360))

([Knoxville News-Sentinel](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:6CGD-XDB1-JB1V-G1DG-00000-00&context=1519360), [Ledger Nashville Edition](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:6BWK-WKF1-DXVP-T0W2-00000-00&context=1519360))

([Jackson Sun](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:6CGM-WG31-JB1V-G1G2-00000-00&context=1519360), [Nashville Post](https://advance.lexis.com/api/document?collection=news&id=urn:contentItem:6BCJ-FF01-JBCN-32YR-00000-00&context=1519360))
